Media Capture Helper
사용 안내 GitHub 다운로드

Windows 10 / 11 · 64비트

PC를 켜두면, 기록과 송출이 자동으로 시작됩니다.

Media Capture Helper는 화면과 소리를 본인 YouTube 채널에 일부 공개로 송출하고, 동시에 PC에 백업 녹화하는 Windows 프로그램입니다. OBS를 따로 열거나 매번 방송을 시작할 필요가 없습니다.

Windows용 다운로드 설치·사용 안내 보기

본인 소유 또는 관리 권한이 있는 PC의 기록·방송 목적에만 사용할 수 있습니다.

처음 시작하기

세 단계면 충분합니다

어려운 방송 설정을 먼저 배울 필요가 없습니다. 설치 후 처음 한 번만 확인하면, 다음부터는 PC를 켜는 것으로 시작됩니다.

  1. 다운로드

    위의 Setup.exe 다운로드 버튼을 눌러 최신 설치 파일을 받습니다.

  2. 실행과 동의

    설치 파일을 실행한 뒤, 화면·소리 캡처와 본인 YouTube 송출에 관한 안내를 읽고 동의합니다.

  3. 최초 연결

    처음 송출할 때만 브라우저에서 본인 Google 계정으로 로그인합니다. 이후에는 자동으로 연결됩니다.

필요한 일만

복잡한 조작 없이, 필요한 기능은 갖췄습니다

↗

자동 송출

PC가 켜진 뒤 준비가 완료되면 YouTube 라이브가 자동으로 시작됩니다. 라이브는 기본적으로 일부 공개로 만들어집니다.

▣

로컬 백업 녹화

인터넷 연결이 불안정해도 PC 녹화는 계속됩니다. 송출이 복구되면 자동으로 다시 연결합니다.

⌘

필요할 때만 제어

Ctrl+Shift+F12로 제어판을 열어 상태, 오디오, 녹화와 설정을 확인할 수 있습니다.

!

설치할 때 Windows 보안 경고가 보일 수 있습니다

개인 배포 프로그램은 서명이 없으면 Windows가 경고를 표시할 수 있습니다. GitHub 공식 릴리스에서 받은 파일인지 확인한 뒤, 경고 화면에서 [추가 정보] → [실행]을 선택하면 설치할 수 있습니다. 자세한 화면 안내는 사용 안내에서 확인하세요.

자주 묻는 질문

다운로드 전 확인해 주세요

YouTube 라이브는 누구나 볼 수 있나요?

기본값은 일부 공개(Unlisted)입니다. 링크를 알고 있는 사람만 볼 수 있고, 공개 검색 결과에는 표시되지 않습니다.

인터넷이 끊기면 녹화도 멈추나요?

아닙니다. 네트워크나 YouTube 연결에 문제가 있어도 로컬 녹화는 계속되고, 연결이 복구되면 송출을 다시 시도합니다.

처음 연결이 바로 되지 않으면 어떻게 하나요?

YouTube 채널의 라이브 스트리밍이 활성화되어 있는지 먼저 확인하세요. 최초 활성화에는 최대 24시간이 걸릴 수 있습니다. YouTube 연결 안내 보기

설치 후 설정을 바꾸고 싶어요

제어판에서 녹화 폴더, 보존 기간, 자동 시작 방식 등을 바꿀 수 있습니다. 교시 VOD나 폰 원격 제어 같은 상세 설정은 전체 사용 안내를 참고하세요.

준비가 되셨나요?

최신 Windows 설치 파일을 다운로드하거나, 설치 전에 전체 사용 방법을 먼저 확인할 수 있습니다.
