Media Capture Helper
다운로드 GitHub 다운로드

사용 안내

설치부터 필요한 설정까지, 순서대로 안내합니다.

Media Capture Helper는 PC 화면과 소리를 본인 YouTube 채널에 일부 공개로 송출하고, 같은 내용을 로컬에도 백업 녹화합니다. 처음에는 설치와 YouTube 연결만 확인하면 됩니다.

설치·첫 실행 일상 사용·제어판 녹화·교시 VOD 폰 원격 제어 설정·관리 문제 해결·이용 안내

1. 설치·첫 실행

처음 한 번만 차례대로 확인하세요

설치가 끝난 뒤에는 PC를 켜는 것만으로 송출과 녹화가 시작됩니다. YouTube 로그인과 캡처 동의는 최초 1회에만 필요합니다.

운영체제Windows 10 또는 Windows 11, 64비트 전용입니다.
인터넷송출과 업로드에는 필요합니다. 연결이 끊겨도 로컬 녹화는 계속됩니다.
YouTube 채널라이브 스트리밍이 활성화된 본인 채널이 필요합니다. 최초 활성화에는 최대 24시간이 걸릴 수 있습니다.
녹화 저장 공간기본 보존 한도는 100GB입니다. 여유 공간이 부족하면 녹화를 일시 중지할 수 있으므로 넉넉한 드라이브를 권장합니다.

설치 순서

  1. Windows 설치 파일을 다운로드해 실행합니다.
  2. 화면에 보이는 사용 안내와 동의 내용을 읽고 진행합니다.
  3. 설치가 끝나면 프로그램이 실행되고, 화면·오디오 캡처와 본인 YouTube 채널 송출에 대한 안내가 한 번 표시됩니다.
  4. 처음 송출할 때 열리는 브라우저에서 본인 Google 계정으로 로그인합니다. 이후 인증 정보는 이 PC에서 안전하게 저장·갱신됩니다.
!

Windows 보안 경고가 보일 때

서명되지 않은 개인 배포 프로그램은 Windows SmartScreen 경고가 표시될 수 있습니다. GitHub 공식 릴리스에서 받은 파일인지 확인한 뒤 [추가 정보] → [실행]을 선택하세요.

YouTube 연결

처음 송출할 때만 브라우저가 열립니다. 본인 Google 계정으로 로그인한 뒤, 사용할 채널을 선택하면 됩니다. 채널의 YouTube Studio → 라이브 스트리밍이 미리 활성화되어 있어야 하며, 처음 활성화하는 경우 반영까지 최대 24시간이 걸릴 수 있습니다.

i

무엇에 동의하나요?

주 모니터 화면과 시스템 소리·마이크를 캡처해 본인 YouTube 채널로 일부 공개 송출하고, 같은 내용을 로컬 폴더에 백업 녹화한다는 내용입니다. 동의하지 않으면 프로그램은 시작하지 않습니다.

2. 일상 사용·제어판

평소에는 PC를 켜는 것만으로 충분합니다

준비가 완료되면 약 30초 뒤 자동으로 송출과 녹화가 시작됩니다. 라이브는 기본적으로 일부 공개(Unlisted)로 생성되어 링크를 아는 사람만 시청할 수 있습니다.

상태 확인

화면 왼쪽 위의 작은 상태 점으로 현재 상태를 빠르게 확인할 수 있습니다.

초록: 정상 송출 중 노랑: 연결·준비 중 빨강: 오류·재시도 중, 녹화는 계속
  • PC를 종료하면 방송이 종료되고 현재 녹화 파일이 안전하게 마감됩니다.
  • 네트워크가 잠시 끊겨도 녹화는 계속되며, 연결이 복구되면 송출을 자동으로 다시 시도합니다.
  • 라이브 URL은 알림을 받은 사람에게만 공유하세요.

제어판 열기

Ctrl+Shift+F12를 누르면 제어판을 열 수 있습니다. 평소에는 열 필요가 없지만, 상태 확인이나 세부 설정을 바꿀 때 사용합니다.

상태·성능LIVE/연결 중/재시도 상태, 수동 시작·중지, 업로드 속도, FPS, CPU·GPU 사용량을 확인합니다.
오디오시스템 소리와 마이크의 장치를 따로 고르고, 볼륨·증폭·음소거를 조절합니다.
녹화현재 파일명, 누적 사용량, 남은 디스크 공간과 녹화 폴더를 확인합니다.
설정호실명, 자동 시작 방식, 단축키, 녹화 보존 기간과 용량 한도를 바꿉니다.
송출 품질과 자원 사용량은 어떻게 조절되나요?

기본값은 네트워크와 컴퓨터 상태에 맞춰 품질을 자동 조절합니다. 제어판에서 표준·절약 같은 품질 단계를 선택하거나 CPU/GPU 자원 사용 한도를 정할 수 있습니다. 제한 변경은 다음 세션부터 적용될 수 있습니다.

3. 녹화·교시 VOD

녹화본은 PC에 남고, 필요하면 교시별로 업로드됩니다

송출과 로컬 녹화는 함께 시작하지만, 네트워크 문제로 송출이 멈춰도 녹화는 독립적으로 이어집니다.

기본 위치%USERPROFILE%\Videos\MediaCaptureHelper — 제어판에서 변경할 수 있습니다.
파일 구성부팅부터 종료까지가 하나의 파일이며, 인코더가 다시 시작되면 _part2 파일이 추가될 수 있습니다.
자동 정리기본 7일이 지난 파일과 100GB 한도를 넘는 오래된 파일부터 자동으로 정리합니다. 두 값은 변경할 수 있습니다.
안전 장치여유 디스크 공간이 5GB 미만이면 녹화를 일시 중지합니다. 갑작스러운 전원 차단 뒤에도 파일을 재생할 수 있도록 처리합니다.

호실명으로 여러 PC의 제목 구분하기

여러 PC가 같은 YouTube 채널로 송출한다면 제어판 설정 → 호실명에 예를 들어 201호를 입력하세요. 이후 라이브와 교시 VOD 제목 앞에 [201호]가 자동으로 붙습니다. 하나의 공유 채널을 쓸 때는 브랜드 계정을 만들고 각 운영자 계정에 관리자 권한을 부여하는 방식을 권장합니다.

교시 자동 분할 VOD 업로드 설정

하루 종일 이어지는 라이브는 그대로 두고, 녹화본만 시간표에 맞춰 교시별로 잘라 일부 공개 VOD로 자동 업로드할 수 있습니다. 쉬는 시간은 업로드하지 않으며, 라이브·녹화 파이프라인을 다시 인코딩하지 않습니다.

  1. 폰 원격 제어에서 요일별 기본 시간표를 입력합니다. 단축수업일에는 그날만 덮어쓰는 시간표를 사용할 수 있습니다.
  2. 교시가 끝나면 녹화본의 해당 구간을 무손실로 잘라 업로드 대기열에 넣습니다. 키프레임 기준 때문에 시작점이 최대 약 2초 차이 날 수 있습니다.
  3. VOD 제목의 기본 형식은 [{호실}] {교시}교시 - {yyyy-MM-dd}입니다. 호실명이 비어 있으면 그 부분은 표시되지 않습니다.
!

YouTube 업로드 한도

YouTube Data API의 기본 일일 한도에서는 영상 업로드가 약 6건까지 가능합니다. 7교시 이상인 날의 초과분은 대기열에 보존됐다가 다음 날 자동 업로드됩니다. 모든 교시를 당일에 올려야 한다면 Google Cloud Console에서 할당량 증설을 신청하세요.

4. 폰 원격 제어

폰 브라우저로 상태를 확인하고 시간표를 편집할 수 있습니다

같은 Wi‑Fi 또는 Tailscale 네트워크에서 폰 브라우저로 접속해 라이브 시작·중지, 상태와 업로드 진행 확인, 교시 시간표 입력을 할 수 있습니다. 폰에 별도 앱을 설치할 필요는 없습니다.

같은 네트워크에서 연결하기

  1. 제어판 설정에서 폰 원격 제어를 켭니다. 기본 LAN 포트는 8787입니다.
  2. 폰 브라우저에서 http://<PC의 LAN IP>:8787로 접속합니다. PC IP는 제어판 로그 또는 ipconfig에서 확인할 수 있습니다.
  3. 제어판에 표시된 6자리 PIN을 입력하면 연결됩니다. 한 번 연결한 폰은 다음부터 PIN 없이 접속할 수 있습니다.
i

외부 접속은 Tailscale을 권장합니다

LAN 모드는 같은 Wi‑Fi 안에서만 쓰세요. 집 밖에서 접속해야 한다면 PC와 폰을 같은 Tailscale 네트워크에 연결하고 remote.mode를 tailscale로 바꾸는 방식을 권장합니다. 공인 IP 직접 노출이나 포트 포워딩은 권장하지 않습니다.

Tailscale로 집 밖에서 연결하기
  1. PC와 폰에 Tailscale을 설치하고 같은 계정으로 로그인합니다.
  2. %AppData%\MediaCaptureHelper\config.json의 remote.mode를 tailscale로 바꾼 뒤 프로그램을 다시 시작합니다.
  3. 폰 브라우저에서 http://<PC의 Tailscale IP>:8787로 접속해 PIN을 입력합니다.
고정 HTTPS 주소가 필요한 경우: Cloudflare Tunnel

매번 같은 외부 주소가 필요하다면 Cloudflare Tunnel을 사용할 수 있습니다. 토큰은 민감 정보이므로 공유하거나 공개 저장소에 올리지 마세요.

  1. Cloudflare에 로그인하고 사용할 도메인을 연결하거나 등록합니다.
  2. Zero Trust → Networks → Tunnels에서 새 터널을 만들고 커넥터로 Cloudflared를 선택합니다.
  3. 터널의 공개 호스트 이름을 추가해 서비스 유형은 HTTP, 대상은 localhost:8787로 지정합니다.
  4. 발급된 터널 토큰과 고정 호스트 이름을 아래처럼 설정 파일에 입력한 뒤 프로그램을 다시 시작합니다.
  5. 외부 브라우저에서 고정 HTTPS 주소로 접속하고, 처음 한 번만 제어판의 6자리 PIN을 입력합니다.
"remote": {
  "mode": "cloudflare",
  "port": 8787,
  "cloudflareTunnelToken": "Cloudflare에서_받은_터널_토큰",
  "cloudflareHostname": "remote.example.com"
}

5. 설정·관리

필요한 설정은 제어판에서, 고급 설정은 파일에서

대부분의 값은 제어판과 폰 원격 화면에서 바꾸면 됩니다. 직접 설정 파일을 편집할 때는 프로그램을 종료한 뒤 UTF-8 형식으로 저장하세요.

알림 받기 (선택)

텔레그램 알림을 설정하면 마이크 음소거, 송출 끊김, 디스크 공간 부족, 업로드 실패 같은 문제를 바로 받을 수 있습니다. Telegram의 @BotFather에서 봇 토큰을 만들고, 알림을 받을 채팅 ID와 함께 제어판 설정에 입력한 뒤 테스트 알림으로 확인하세요. 봇 토큰은 비밀번호처럼 취급해야 합니다.

config.json 주요 항목

설정 파일은 %AppData%\MediaCaptureHelper\config.json에 있습니다. 시간표는 폰 원격 화면에서 입력하면 자동으로 채워집니다.

{
  "deviceName": "201호",
  "youtube": { "titleTemplate": "[{호실}] 라이브 - {yyyy-MM-dd HH:mm}" },
  "periods": {
    "weekdayDefaults": { "Mon": [{ "no": 1, "start": "09:00:00", "end": "09:50:00" }] },
    "overrides": { "2026-07-14": [{ "no": 1, "start": "09:00:00", "end": "09:30:00" }] },
    "titleTemplate": "[{호실}] {교시}교시 - {yyyy-MM-dd}",
    "vodPrivacy": "unlisted",
    "uploadTiming": "immediate-throttled"
  },
  "remote": { "mode": "off", "port": 8787 }
}

uploadTiming은 기본값인 immediate-throttled 외에, 방과 후 일괄 업로드를 위한 after-hours로 바꿀 수 있습니다.

자동 업데이트

프로그램은 주기적으로 새 버전을 확인합니다. 다운로드된 업데이트는 보통 다음 재시작 때 적용되므로, 진행 중인 송출을 즉시 끊지 않습니다.

제거

Windows 설정 → 앱에서 Media Capture Helper를 제거합니다. 자동 시작과 원격 방화벽 규칙은 함께 해제되며, 녹화 파일과 설정은 자동 삭제되지 않습니다.

6. 문제 해결·이용 안내

문제가 생겨도 먼저 이 항목을 확인하세요

상태 점이 계속 노란색이에요

인터넷 연결과 YouTube 라이브 스트리밍 활성화 상태를 확인하세요. 채널을 처음 활성화했다면 최대 24시간 동안 준비 중일 수 있습니다.

상태 점이 계속 빨간색이에요

%AppData%\MediaCaptureHelper\logs\의 최신 로그를 확인하세요. 프로그램은 자동으로 재연결을 계속 시도하며, 로컬 녹화는 유지됩니다.

Google 로그인 브라우저가 열리지 않아요

%AppData%\MediaCaptureHelper\client_secret.json 파일이 있는지 확인하세요. 파일의 내용이나 OAuth 토큰을 타인에게 공유하지 마세요.

마이크 소리가 들리지 않아요

제어판의 오디오 영역에서 마이크 장치와 볼륨, 음소거 상태를 확인하세요. 마이크가 없어도 시스템 소리는 별도로 송출할 수 있습니다.

송출은 끊겼는데 녹화는 계속돼요

네트워크 문제가 있을 때 예상되는 동작입니다. 인터넷이 복구되면 송출을 자동으로 다시 시도하며, 그동안 로컬 녹화본은 계속 남습니다.

게임 전체 화면이 검게 나와요

자동 재초기화가 끝날 때까지 기다리거나, 게임을 창 모드 또는 테두리 없는 전체 화면으로 바꿔 보세요.

단축키가 작동하지 않아요

다른 프로그램이 같은 조합을 사용하고 있을 수 있습니다. 제어판 설정에서 다른 단축키로 변경하세요.

이용 안내

!

본인 소유 또는 관리 권한이 있는 PC에서만 사용하세요

이 도구는 본인의 화면과 음성을 기록·방송하는 용도로만 사용합니다. 타인의 화면·음성·개인정보를 동의 없이 캡처하거나 감시·스토킹 목적으로 사용하는 것은 금지됩니다.

전체 이용 조건 요약
  1. 동작: 주 모니터 화면과 시스템·마이크 오디오를 캡처해 사용자가 직접 연결한 YouTube 채널로 일부 공개 송출하고, 로컬에 백업 녹화합니다.
  2. 데이터: 녹화 파일은 사용자가 지정한 로컬 폴더에 저장됩니다. OAuth 토큰은 이 PC에서 보호되며, 개발자에게 자동 전송되지 않습니다.
